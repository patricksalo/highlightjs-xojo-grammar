# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0]

Initial release.

### Added

- Added Xojo language grammar for highlight.js.
- Added source grammar at `src/languages/xojo.js`.
- Added compiled browser build at `dist/xojo.min.js`.
- Added compiled ES module build at `dist/xojo.es.min.js`.
- Added basic automated tests for Xojo syntax highlighting.
- Added browser-based visual test page at `test/index.html`.
- Added README with usage notes for tools that support custom highlight.js grammars.
- Added MIT license.
