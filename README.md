# Xojo grammar for highlight.js

Xojo is a cross-platform programming language and development environment for building native desktop, mobile, web, and console applications.

This repository provides a Xojo language grammar for highlight.js so Xojo source code can be highlighted in tools, documentation, and websites that support highlight.js.

Repository: https://github.com/patricksalo/highlightjs-xojo-grammar

For more about Xojo, visit: https://www.xojo.com/.

## Usage

Copy or bundle the grammar into tools that support custom highlight.js language definitions. Load the Xojo grammar after loading highlight.js, then register or use it according to the host application's integration model.

For applications such as Kaleidoscope, follow the host application's instructions for adding a custom highlight.js grammar.

## Tests

Install development dependencies and run the test suite:

```bash
npm install
npm test
```
